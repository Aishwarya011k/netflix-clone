# Netflix Clone (HTML, CSS, JS)

This project is a clone of the Netflix main page, developed using vanilla HTML, CSS, and JavaScript. It aims to recreate the look-and-feel of the Netflix homepage and simulate some of its UI interactions.

## Features

- Responsive Netflix-inspired homepage design
- Movie/Show cards grid
- Navbar with logo, navigation links, and profile dropdown (static or interactive)
- Hover effects on cards
- Sections for Trending, Originals, etc. (as seen on Netflix)
- Basic Modal or video popup (if implemented)
- Search bar (static or functional search)
- Footer section

## Technologies Used

- **HTML**: Page structure and layout
- **CSS**: Styling, grid/flex layout, animations, responsiveness
- **JavaScript**: Basic interactivity (dropdowns, modals, sliders, search, etc.)

## How to Run Locally

1. **Clone or download the repository:**
   ```bash
   git clone https://github.com/Aishwarya011k/netflix-clone.git
   cd netflix-clone
   ```

2. **Open `index.html` in your web browser.**

No build tools or installations required!
